Full Stack Web Development Tech Stack:

HTML
HTML stands for Hyper Text Markup Language. it is a standard markup language for creating web pages and other web documents. HTML defines the content and structure of web content. HTML consists of a series of elements. HTML elements tell the browser how to display the content.
CSS
CSS stands for Cascading style sheet.it is a style sheet language used for explaining the presentation of a document written in HTML or XML. CSS is unique in that it doesnot create any new elements like HTML or Javascript.CSS has the ability for separation of content and style,consistency,ease of maintainance, accessibility etc.
JavaScript
JavaScript is a programming language used to create interactive effects within web browser.it allows developers to create dynamically updating content,use animation ,pop-up menus,clickable buttons,control multimedia. JavaScript can change HTML style.Javascript can hide html elements.hiding html elements can be done by changing the display style.
Node.js
Node.js is an open-source, cross-platform, JavaScript runtime environment that executes JavaScript code outside of a browser. Node.js are Javascript technologies basically it is a framework of Javascript which is mainly used for working with the backend of our application or building the backend using JavaScript.
Express.js
Express.js is a web application framework for Node.js designed for building web applications and APIs.Express.js is an extensive part of the JavaScript ecosysytem.it is designed to build single page,multipage and hybrid web application. Express.js is used for a wide range of things in the JavaScript or node.js ecosysytem.
MongoDB
MongoDB is a NoSQL document-oriented database program that uses JSON-like documents with optional schemas.MongoDB can operate as a generall data store.MongoDB offers support for many programming languages sucha as C,C++,C#,GO,JAVA,PYTHON,RUBY and SWIFT.
React.js
React.js is a front-end declarative , effecient and flexible JavaScript library for building user interfaces or UI components.React.js is significantly decreases the code with its components, states i.e hooks etc, the main features of React.js that distinguishes it from other popular Javascript framework is flexibility.
